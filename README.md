# nogaraviv.com

Portfolio site for Noga Raviv.

[View nogaraviv.com](http://nogaraviv.com)

## Development

```bash
$ npm install
$ npm start
```

## Deployment

```bash
$ npm run deploy
```

This builds the site in `/bin`, then pushes the contents of `/bin` to the `gh-pages` branch on `origin` (github.com).
